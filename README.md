# Activity 5. Memory Game

> Classic memory card game where players flip tiles to find matching pairs. Built with Python and the `freegames` library using turtle graphics.

Based on [Grant Jenks' freegames cannon](http://www.grantjenks.com/docs/freegames/cannon.html).

# Changes

- Count and display the number of taps
- Detect when all squares have been uncovered
- Center the digit in the square
- As a touch of innovation to the game, you can match patterns by letters and numbers
